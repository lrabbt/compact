## Installation

### Source code

Requirements:

* Python >= 3.8

Clone the repository and enter it:

```sh
git clone git@github.com:lrabbt/compact.git
cd compact
```

Use `pip` for installation:

```sh
pip install .
```

## Usage

For available commands and options:

```sh
compact --help
```

For utils:

```sh
compact-utils --help
```
